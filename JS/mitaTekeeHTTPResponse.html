Mita tekee?

Get - GET:tiä käytetään ottamaan tietoa serveriltä käyttäen annettua URI:a. GET REQUESTien ei tulisi tehdä muuta, kuin noutaa dataa.

Post - POST REQUESTia käytetään lähettamään dataa serverille, kuten asiakkaan tietoja, käyttäen HTML formeja.

Head - Muuten sama, kuin GET paitsi siirtää vain STATUS LINEn ja HEADER osion.

Put - Korvaa kaikki sen hetkiset esitysmuodot kohde resurssista ladatulla sisällöllä.

Delete - Poistaa kaikki sen hetkiset esitysmuodot kohde resurssista annetulla URI:lla.

Connect - Luo tunnelin annetun URIn määrittämään serveriin.

Options - Kuvailee viestintä asetukset kohde resurssille.

Trace - Tekee viestisilmukkatestin matkalla kohde resurssiin.

HTTP Response

1xx: Informational - Tarkoittaa, että REQUEST vastaanotettiin ja prosessi jatkuu.

2xx: Success - Tarkoittaa, että toiminto otettiin vastaan, ymmärrettiin ja hyväksyttiin onnistuneesti.

3xx: Redirection - Tarkoittaa, että pitää suorittaa jatkotoimenpiteitä REQUESTin suorittamiseksi.

4xx: Client Error - Tarkoittaa, että REQUESTissa on virhe ja sitä ei voi suorittaa.

5xx: Server Error - Tarkoittaa, että serveri ei pystynyt suorittamaan kelpaavaa REQUESTia.
